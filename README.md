# shockratees
E-commerce platform for Shockratees apparel brand. Built with Python FastAPI and modern frontend stack.

## GitHub Pages

This repository is prepared for GitHub Pages deployment from the `main` branch using the repository root as the publishing source. The static site entry point is `index.html` at the root, with `styles.css` and `script.js` referenced from the same directory so the page works both through GitHub Pages and when opened directly in a browser.
